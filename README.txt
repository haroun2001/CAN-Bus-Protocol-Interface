# CAN Bus Protocol Interface

## Description
Cette interface web présente le protocole Bus CAN et ses fonctionnalités. Elle permet la génération de trames CAN à partir des données saisies par l'utilisateur et inclut le calcul du CRC.

## Fonctionnalités
- Génération de trames CAN à partir de la partie Data.
- Calcul du CRC pour les trames CAN.
